# 🏗️ AREP: Enterprise Architecture

Welcome to the repository for **AREP** (Arquitectura Empresarial - Enterprise Architecture) at the *Escuela Colombiana de Ingeniería Julio Garavito*.
This comprehensive collection showcases the journey through enterprise and software architecture — from digital transformation and machine learning fundamentals to distributed systems, cloud-native microservices, and the epistemological foundations of computing 🌐.

---

## 📚 Topics Covered

### 🤖 Machine Learning & AI Foundations

* Linear and polynomial regression with gradient descent
* Logistic regression and classification
* Neural networks and convolutional architectures (CNNs)
* LLM embeddings and text preprocessing
* LangChain basics and LLM chain patterns
* Retrieval-Augmented Generation (RAG) with OpenAI and Pinecone

### 🔄 Digital Transformation & Enterprise Architecture

* Introduction to digital transformation concepts
* Enterprise architecture frameworks and principles
* Data-driven and event-driven AI architectures
* Agentic LLM patterns

### 🌐 Distributed Systems Design

* Basic concepts of distributed system design
* Object-oriented design with generics and lambda expressions
* Names, networks, client-server models
* Inversion of Control (IoC) and meta-reflection/annotations

### 🐳 Virtualization & Containerization

* Java threads and concurrency
* Services and virtualization
* Docker fundamentals and modular containerization

### 🔐 Security Architecture

* Security design principles
* Secure application design with Spring
* Auth0 integration and JWT-based authentication

### ☁️ Microservices & Serverless Architectures

* Introduction to microservices
* AWS Lambda functions and API Gateway in Java
* RESTful Web API design
* Microservices on AWS architecture

### 💡 Epistemology, Government & Society

* Science, epistemology, and computing (David Deutsch – *The Beginning of Infinity*)
* A new idea of government and digital governance
* Sustainability and optimism in technology

### 📐 Towards a New Theory of Enterprise Architecture

* Code as design (Reeves)
* Economic principles applied to software projects
* Budgeting and project management in architecture
